Quickstart

Get started with the Fluxfeed API in seconds

Get Latest News 
curl "https://api.fluxfeed.news/v1/news?page=1&page_size=5"
Trending Mentions (7 days) 
curl "https://api.fluxfeed.news/v1/trending/top-mentioned?last_days=7&limit=10"
JavaScript (Fetch API) 
const response = await fetch(
  'https://api.fluxfeed.news/v1/news?page=1&page_size=5'
);
const data = await response.json();
console.log(data.items);
Python (requests) 
import requests

r = requests.get(
    'https://api.fluxfeed.news/v1/trending/headlines',
    params={'limit': 10, 'last_hours': 24}
)
data = r.json()
for item in data['items']:
    print(item['title'])
